# Hệ Thống Quản Lý Vật Tư

## Giới thiệu
Hệ thống này sử dụng công nghệ blockchain để theo dõi và quản lý vật tư trong chuỗi cung ứng. Nó giúp các nhà sản xuất, nhà cung cấp và các bên liên quan khác đảm bảo tính minh bạch và an toàn trong việc giao dịch và quản lý vật tư.

## Tính năng chính
- Theo dõi lịch sử vật tư trong thời gian thực.
- Tạo hợp đồng thông minh để tự động hóa quy trình giao dịch.
- Cung cấp báo cáo minh bạch về nguồn gốc và trạng thái của vật tư.
- Tích hợp dễ dàng với các hệ thống quản lý hiện có.

## Công nghệ sử dụng
- Blockchain (Ethereum hoặc Hyperledger)
- Web3.js hoặc ethers.js cho frontend
- Node.js cho backend

## Cài đặt
1. Clone repository về máy: `git clone https://github.com/username/he-thong-quan-ly-vat-tu`
2. Cài đặt các thư viện cần thiết: `npm install`
3. Chạy ứng dụng: `npm start`
